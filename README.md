# Rota Vialog Treinamentos

Site institucional responsivo desenvolvido com Bootstrap 5, voltado para <b>cursos e capacitações</b> na <b>área de Logística</b> <br>

<b>Acesse aqui:</b> https://elisdevweb.github.io/projeto-rota-vialog-treinamentos/

##  Funcionalidades
- Estrutura organizada com Bootstrap 5
- Seção “Sobre” com apresentação da empresa
- Seção “Cursos” com cards uniformes e hover animado
- Formulário de contato com validação em JavaScript
- Layout totalmente responsivo (desktop, tablet e mobile)
- Banners com efeito parallax e overlay escuro para contraste
- Código limpo e otimizado para melhor desempenho
- Navbar interativa com destaque automático da seção ativa
- Rolagem suave entre seções

##  Tecnologias Utilizadas
- **HTML5** → Estrutura semântica das páginas  
- **CSS3** → Estilização visual 
- **Bootstrap 5** → Componentes e grid responsivo  
- **JavaScript** → Interatividade e controle da navbar/formulário

##  Estrutura do Projeto

│── index.html # Página principal do site <br>
│── style.css # Estilos globais e variáveis de cores <br>
│── script.js # Interatividade da navbar e validação do formulário <br>
│── /assets # Imagens, ícones e banners <br>
│── /bootstrap # Biblioteca Bootstrap
